== Installation ==

1. Download the plugin woo-onepage-checkout.zip file. Extract and upload in your wp-content/plugins folder.
2. Activate the plugin through the 'Plugins' screen in WordPress by clicking 'Activate' button below 'Checkout Add-on for Woo OnePage -Lite'.
3.Use the OnePage Shop->Settings screen to configure the your affiliate account tag and country.
4. You are done.
